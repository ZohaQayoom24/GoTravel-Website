# Go Travel - Travel Agency Website

A modern and responsive travel agency website that helps users explore destinations, check weather conditions, and book their dream trips.

## Features

- **Home Page** - Welcome page with hero section
- **About** - Information about Go Travel agency
- **Destinations** - Browse available travel destinations
- **Testimonials** - Customer reviews and experiences
- **Gallery** - Photo gallery of destinations
- **Weather Check** - Real-time weather information for travel planning
- **Contact** - Get in touch with the agency
- **Booking System** - Easy trip booking functionality
- **User Authentication** - Sign up and login features

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Responsive Design (Mobile-friendly with hamburger menu)

## Installation

1. Clone the repository
```bash
git clone https://github.com/your-username/go-travel.git
```

2. Open `index.html` in your browser
